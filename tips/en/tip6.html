Is something broke?

You can do the following if you think Temp_Cleaner GUI isn't working as it should.

  • Restarting Temp_Cleaner GUI
    • Close the program by closing the main window.
    • Open Task manager: Ctrl + Shift + Esc
    • Look for any open instances of Temp_Cleaner GUI
    • Right click on the instance then click Kill task
  • After you restart Temp_Cleaner GUI, try to do what you did when the issue appeared, and see if it was fixed.
  • Create an issue in Temp_Cleaner GUI's official Github repository
    • Open: (Github repository: InsertX2k/temp_cleaner_gui) 'https://github.com/insertx2k/temp_cleaner_gui' on any web browser
    • Click on the "issues" button
    • Click "New issue"
    • Write a short title describing your issue and in the description, Please write the steps to reproduce the issue and what exactly you were expecting Temp_Cleaner GUI to do that time and the version of Temp_Cleaner GUI you are running along with any other information you think might be useful in identifying the problem (Like your OS version, Architecture, the amount of RAM you have on your PC, Processor speed and model and name), it would also be really helpful if you included a screenshot of Temp_Cleaner GUI's window.
    • Click "Submit new issue", You may be asked to login or create a new account in Github if you already haven't.
  • If you don't feel like creating issues on Github, you can open a support ticket in Temp_Cleaner GUI's Official sourceforge page
    • Open: 'https://sourceforge.net/p/temp-cleaner-gui/tickets/' on any web browser
    • Click "Create ticket"
    • Follow on-screen instructions, it may also ask you to create an account in sourceforge if you already haven't
  • If none of these options seem to be suitable for you, You may then contact the support of Temp_Cleaner GUI
    • Open: 'https://twitter.com/insertplayztw' and send a direct message there describing the issue you are facing and steps for reproducing the issue and other information you think will be useful in identifying the issue.

      • or alternatively:

    • Join our support discord server (Go to Temp_Cleaner GUI's official website) and look for the support server link and join it and send a message in the main-chat channel describing the issue you are facing along with any other information you think will be useful in identifying the issue.
