Got lost?

Did you know that Temp_Cleaner GUI provides you with a useful help document that covers the basics of using it, it also has a section for first time users.

Opening Help document, the usual way:

  • Open Start menu
  • Click "All Programs" (If you are running Windows 8.1 then you need to right click on any empty space on your screen so you can see the option to show all programs first)
  • Look for a folder named "Ziad (Mr.X) Software"
  • Open it
  • Click on the shortcut named "Temp_Cleaner GUI Help"

If you can't open start menu or you don't find that shortcut for some reason, you can still open the help document manually yourself

  • Open the directory where Temp_Cleaner GUI is installed in (for example: C:\Program Files\Temp_Cleaner GUI)
  • Open the folder "help_docs"
  • Open the file "index.html"
