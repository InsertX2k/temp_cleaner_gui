Keeping Temp_Cleaner GUI up to date

Temp_Cleaner GUI provides you with an easy and comfortable way to keep it up to date, I will be listing the official 3 methods to check for updates.

Via the built-in Updater program

Temp_Cleaner GUI provides you with a built-in updater program, which allows you to automate the process of checking for updates.

Starting the built-in updater program can be done in two ways

  • When automatic updates check is enabled in settings, Temp_Cleaner GUI will automatically start checking for updates whenever you launch it, and when an update is available, Temp_Cleaner GUI will launch the updater window informing you of a new update.
  • You can launch the built-in updater program if you have the automatic updates check option disabled or you just want to check for updates manually, you may start the built-in updater program by following these instructions:
    • Launch Temp_Cleaner GUI
    • Scroll down til there is nothing left and look for the About (may vary depending on the UI language) button
    • Click on it
    • Click on the button Update Check (may vary depending on UI language)
    • If an update is available, the updater program will launch, showing you the release notes of the latest release and the version number of the latest update.

      and if there isn't, A message box will appear on your screen telling you that you are up to date.

Manually by going to the Temp_Cleaner GUI's website

You may manually check for (and download) updates from the official Temp_Cleaner GUI's website, this can be useful in case you are running an older version of Temp_Cleaner GUI or your firewall (or antimalware) program blocked Temp_Cleaner GUI from accessing the internet or when you can't access internet on your PC.

  • Launch Temp_Cleaner GUI
  • Scroll down til there is nothing left and look for the About (may vary depending on the UI language) button
  • Go to the official downloads page of Temp_Cleaner GUI (this is a hyperlink by the way)
  • Compare the version in the About window with the one in that website.
  • If the version in the website is newer than (number is higher than) the one in the About window, then you need to update your release of Temp_Cleaner GUI.
    • Click on the appropriate download button on the downloads page according to your architecture and operating system version
    • Wait for the file to finish downloading
    • Uninstall the previous installation of Temp_Cleaner GUI (recommended but not obligatory), See Uninstalling Temp_Cleaner GUI and removing all of its traces
    • Double click (Run) the file you just downloaded
    • Follow on-screen instructions to complete the installation process.


    Otherwise, if the version in the website is equal to the one in the About window, then you are up to date and you don't need anything else.
