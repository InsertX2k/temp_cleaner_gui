Temp_Cleaner GUI Help

Keyboard Shortcuts & hidden functions

Pretty much like any other piece of software, Temp_Cleaner GUI allows you to quickly access different functions from Keyboard shortcuts.

Keyboard shortcuts for changing font size (of the widget that shows what was cleaned)

  • Ctrl + I - For Increasing font size
  • Ctrl + D - For decreasing font size

Other keyboard shortcuts

  • F2 - to show you a small window (messagebox) containing some useful information about your Windows drive (or volume).
  • F1 - to show you information about using the minimal accessibility pack (or basically increasing or decreasing font size of the widget that shows what was cleaned).
