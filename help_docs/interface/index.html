Temp_Cleaner GUI Help

Understanding the Interface of Temp_Cleaner GUI

Temp_Cleaner GUI, features an easy to use and an easily understandable and far from complication graphical user interface (GUI)

Below is a table containing more information about what each number means

Number Information
#1 The cleaning preset chooser combobox (Dropdown)
Click on the down arrow button on the right (or left side, according to your UI language) to choose a cleaning preset.
A cleaning preset is an option that allows you to automatically select multiple cleaning options just by choosing how you want this program to work.

Usually the most selected preset for everyday cleaning is:
  • Default Preset (Recommended for Everyday use)
More cleaning presets are also available:
  • Maximum Cleaning (Cleans up all sections, thus giving you more disk space)
  • Empty Recycle Bin Only
  • Web Browser Cleaning
  • Web Browser Cleaning + Cookies
  • Cancel Pending Windows Update
First time users (those who probably has their PC hogged with tons of Temporary files) tend to choose this preset:
  • Maximum Cleaning (Cleans up all sections, thus giving you more disk space)
#2 Available cleaning options.
Every cleaning option in Temp_Cleaner GUI is basically a checkbox, as you probably hinted, all you need to do is just checking (clicking) the option you want to clean.
This puts the selected option in the queue of options to clean, and that option will be cleaned when you start the cleaning process.
#3 Window title
This is the title of Temp_Cleaner GUI's Window
This usually shows "Welcome [username] to Temp_Cleaner GUI v[version number]!"
and if it fails to get the current user name, the title will be only "Welcome to Temp_Cleaner GUI v[version number]!"
#4 Banner image
This is basically a banner that sits on top of even the first widget, I put it here because it makes the program looks prettier.
This banner image is changable (can be changed by you)
There are two variants of the file of this banner image:
  • banner.png and this is the default banner file that will appear when you are using Temp_Cleaner GUI on a display with a resolution of 1152x864 or higher.
  • bannerlowres.png and this is the banner file that will appear when you are using Temp_Cleaner GUI on a display with a resolution of 1024x768 or 800x600.
So basically, if you need to modify the banner, you will need to edit one or all of these files which will be stored in the installation folder of Temp_Cleaner GUI. (for example: C:\Program Files\Temp_Cleaner GUI)

Edit them as they are (don't change their resolution or file extension or bit depth or file name)

When you scroll down (using mouse wheel or the scrollbar on the right or left side of the window, depending on the UI language)
You will see the following new interface elements (widgets).

I will also explain what each button and thing does:

Number Information
#1 The "Start cleaning" (may vary between other UI languages) button
This button is probably the most used button in the history of this program, basically because of what it does, it simply executes the cleaner to clean all the options you checked previously
This is the button you will need to click after you choose a cleaning preset or check a cleaning option.

After the cleaning process is complete, Temp_Cleaner GUI will show you a small window (named a messagebox) showing you how many disk space you have just freed up.
#2 The "About & Tips" (may vary between other UI languages) button
When you click this button, it will basically spawn another window in front of you that shows you basic information about your current Temp_Cleaner GUI installation, such as current version, number of contributors to this version, release channel, and support information.
It also allows you to use various functions, like:
  • Showing the license of Temp_Cleaner GUI
  • Opening Temp_Cleaner GUI's official website
  • Checking for program updates
  • Showing a list of all donators to this program
#3 The "Settings" (may vary between other UI languages) button
It basically launches the settings window that allows you to configure Temp_Cleaner GUI the way you like.
#4 This is a widget showing you what has been cleaned and what hasn't.
You can clear that list (delete all text written in it) by pressing the F6 key on your keyboard.
